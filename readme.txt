Gravity form for wordpress, build form with various type of fields like, file,image upload,Date,Select,radio,checkbox and text fields.

== Description ==

[youtube http://www.youtube.com/watch?v=9arMo2Pg7i0]

Build any type of form with various type of input fields and collect data from any page or post in the wordpress.

1. Build form and display via short code or through php function.
2. Captcha enabled to avoid spammers.
3. Token based form to avoid spammers.
4. Date scheduling for form.
5. Stores form data in any type of post type. It creates meta field automatically if you have not created.
6. Support image upload and file upload.
7. Support date field with jquery date picker.
8. Full customizable form with css and html.
9. enable/disable form any time.
10. Arrage fields order by drag n drop.
11. We recommends <a href="https://wordpress.org/plugins/simple-content-construction-kit/">Custom post type creater</a> to create new custom post type for this plugin.

== Installation ==

1. Upload `ultra-gravity-form` to `/wp-content/plugging/` directory
2. Activate plugging through the 'Plugins' menu in WordPress
3. Click on Ultra Gravity Form.
